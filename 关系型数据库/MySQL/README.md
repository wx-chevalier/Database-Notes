# MySQL

针对不同的业务场景选择合适的 SQL/NoSQL/NewSQL 数据库，微服务，云原生的场景下，合理的数据建模，选择合理的字段，索引，外键等，这也是本章讨论的重点。

对于 MySQL 数据库的优化，请参阅本系列的[拆分与调优](../../拆分与调优/README.md)章节。
